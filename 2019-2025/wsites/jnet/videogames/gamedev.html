[ Voidfighter ]

Voidfighter is a 2D Shmup game (shoot-em-up), Control a cat facing off against some crystalline alien entities in the void. An infinite never-ending level allows players to fight for the highscore on their local machine!

Voidfighter was developed for me to learn the C programming language along with the SDL libraries to allow the program to be playable on the platform/os it is ported to.
